# Customer Churn Prediction

Machine learning project to predict customer churn using Python and Scikit-learn.

## Project Overview
This project uses classification models to predict whether a customer is likely to cancel a subscription-based service.

## Dataset
The project uses the Telco Customer Churn dataset from Kaggle.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
